# 03 · HashTable 哈希表

## 🎯 章节核心

哈希表 = **快速判断元素是否存在 / 重复**。空间换时间。

三种结构选型：

| 结构 | 适用 | 缺点 |
|---|---|---|
| **数组** | 数值范围有限（如 26 字母） | 范围大时浪费 |
| **set** | 只关心是否存在 | 不能存额外信息 |
| **map** | 需要 key→value 关联 | 实现稍重 |
