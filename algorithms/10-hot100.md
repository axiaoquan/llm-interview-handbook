### 49. 字母异位词分组

> [LeetCode 49](https://leetcode.cn/problems/group-anagrams/)

#### 🎯 思路

判断异位词的标准：**26 长度的字符计数 tuple** 作为 key。

```python
from collections import defaultdict

class Solution:
    def groupAnagrams(self, strs):
        d = defaultdict(list)
        for s in strs:
            cnt = [0] * 26
            for c in s:
                cnt[ord(c) - ord('a')] += 1
            d[tuple(cnt)].append(s)        # list 不能 hash，转 tuple
        return list(d.values())
```

> 也可以用 `''.join(sorted(s))` 当 key，思路一样。

### 128. 最长连续序列

> [LeetCode 128](https://leetcode.cn/problems/longest-consecutive-sequence/)

#### 🎯 思路（关键 trick）

先用 set 去重；遍历元素，**只从"序列起点"开始计数**（即 `x-1` 不在 set 里）。

```python
class Solution:
    def longestConsecutive(self, nums):
        s = set(nums)
        ans = 0
        for x in s:
            if x - 1 in s: continue        # 只从起点算
            cur = x
            while cur + 1 in s: cur += 1
            ans = max(ans, cur - x + 1)
        return ans
```

**O(n)**。如果不加"只从起点"判断会变成 O(n²)。

## 双指针

### 283. 移动零

> [LeetCode 283](https://leetcode.cn/problems/move-zeroes/)

快慢指针：fast 找到非 0 就和 slow 交换。

```python
class Solution:
    def moveZeroes(self, nums):
        slow = 0
        for fast in range(len(nums)):
            if nums[fast] != 0:
                nums[slow], nums[fast] = nums[fast], nums[slow]
                slow += 1
```

### 11. 盛最多水的容器

> [LeetCode 11](https://leetcode.cn/problems/container-with-most-water/)

#### 🎯 思路

底边由两边短板决定。双指针从两端向中间，**每次移动短板那一边**（底边只能缩小，要让面积变大只能换短板）。

```python
class Solution:
    def maxArea(self, h):
        l, r, ans = 0, len(h) - 1, 0
        while l < r:
            ans = max(ans, min(h[l], h[r]) * (r - l))
            if h[l] < h[r]: l += 1
            else: r -= 1
        return ans
```

## 滑动窗口

### 3. 无重复字符的最长子串

> [LeetCode 3](https://leetcode.cn/problems/longest-substring-without-repeating-characters/)

#### 🎯 思路

左指针每次右移一格，右指针不断右移。用 set 维护窗口内字符。

```python
class Solution:
    def lengthOfLongestSubstring(self, s):
        seen = set()
        l = ans = 0
        for r, c in enumerate(s):
            while c in seen:
                seen.remove(s[l])
                l += 1
            seen.add(c)
            ans = max(ans, r - l + 1)
        return ans
```

### 438. 找到字符串中所有字母异位词

> [LeetCode 438](https://leetcode.cn/problems/find-all-anagrams-in-a-string/)

固定窗口 + 字符计数比较。

```python
class Solution:
    def findAnagrams(self, s, p):
        if len(s) < len(p): return []
        need = [0] * 26
        for c in p: need[ord(c) - ord('a')] += 1
        cur = [0] * 26
        for c in s[:len(p)]: cur[ord(c) - ord('a')] += 1
        ans = []
        if cur == need: ans.append(0)
        for i in range(len(p), len(s)):
            cur[ord(s[i]) - ord('a')] += 1
            cur[ord(s[i - len(p)]) - ord('a')] -= 1
            if cur == need: ans.append(i - len(p) + 1)
        return ans
```

## 子串

### 560. 和为 K 的子数组

> [LeetCode 560](https://leetcode.cn/problems/subarray-sum-equals-k/)

#### 🎯 注意

数组**含负数**→ **不能用滑动窗口**（无单调性）。
用**前缀和 + 哈希表**：当前 prefix - k 在哈希表中出现几次，就有几个子数组和为 k。

```python
from collections import defaultdict

class Solution:
    def subarraySum(self, nums, k):
        prefix = 0
        cnt = defaultdict(int)
        cnt[0] = 1            # 空前缀
        ans = 0
        for x in nums:
            prefix += x
            ans += cnt[prefix - k]
            cnt[prefix] += 1
        return ans
```

### 76. 最小覆盖子串

> [LeetCode 76](https://leetcode.cn/problems/minimum-window-substring/)

#### 🎯 思路

滑动窗口 + 两个哈希表（need / have）+ 计数变量 `valid` 跟踪满足要求的字符数。

```python
from collections import Counter

class Solution:
    def minWindow(self, s, t):
        need = Counter(t)
        have = {}
        valid = 0
        l = 0
        ans = (float('inf'), 0, 0)
        for r, c in enumerate(s):
            have[c] = have.get(c, 0) + 1
            if c in need and have[c] == need[c]:
                valid += 1
            while valid == len(need):
                if r - l + 1 < ans[0]:
                    ans = (r - l + 1, l, r)
                if s[l] in need and have[s[l]] == need[s[l]]:
                    valid -= 1
                have[s[l]] -= 1
                l += 1
        return s[ans[1]: ans[2] + 1] if ans[0] != float('inf') else ''
```

## 普通数组

### 53. 最大子数组和

> [LeetCode 53](https://leetcode.cn/problems/maximum-subarray/)

#### 🎯 思路（DP / Kadane）

`dp[i]`：以 i 结尾的最大子数组和。

$$
dp[i] = \max(nums[i],\ dp[i-1] + nums[i])
$$

```python
class Solution:
    def maxSubArray(self, nums):
        cur = ans = nums[0]
        for x in nums[1:]:
            cur = max(x, cur + x)
            ans = max(ans, cur)
        return ans
```

### 56. 合并区间

> [LeetCode 56](https://leetcode.cn/problems/merge-intervals/)

按起点排序，能合并就合并。

```python
class Solution:
    def merge(self, intervals):
        intervals.sort()
        ans = [intervals[0]]
        for s, e in intervals[1:]:
            if s <= ans[-1][1]:
                ans[-1][1] = max(ans[-1][1], e)
            else:
                ans.append([s, e])
        return ans
```

### 240. 搜索二维矩阵 II

> [LeetCode 240](https://leetcode.cn/problems/search-a-2d-matrix-ii/)

#### 🎯 思路

从**右上角**出发：

- 当前元素 < target → 向下
- 当前元素 > target → 向左

```python
class Solution:
    def searchMatrix(self, matrix, target):
        if not matrix: return False
        m, n = len(matrix), len(matrix[0])
        i, j = 0, n - 1
        while i < m and j >= 0:
            if matrix[i][j] == target: return True
            if matrix[i][j] < target: i += 1
            else: j -= 1
        return False
```

### 958. 二叉树的完全性检验

> [LeetCode 958](https://leetcode.cn/problems/check-completeness-of-a-binary-tree/)

#### 🎯 思路

层序遍历，**遇到第一个 None 后**，后续不能再有非 None。

```python
from collections import deque

class Solution:
    def isCompleteTree(self, root):
        q = deque([root])
        seen_null = False
        while q:
            node = q.popleft()
            if node is None:
                seen_null = True
            else:
                if seen_null: return False
                q.append(node.left)
                q.append(node.right)
        return True
```

### 322. 零钱兑换

> [LeetCode 322](https://leetcode.cn/problems/coin-change/)

完全背包：每种硬币无限。`dp[j]` = 凑出金额 j 的最少硬币数。

```python
class Solution:
    def coinChange(self, coins, amount):
        dp = [float('inf')] * (amount + 1)
        dp[0] = 0
        for c in coins:
            for j in range(c, amount + 1):
                dp[j] = min(dp[j], dp[j - c] + 1)
        return dp[amount] if dp[amount] != float('inf') else -1
```

### 416. 分割等和子集

> [LeetCode 416](https://leetcode.cn/problems/partition-equal-subset-sum/)

#### 🎯 思路

01 背包：能否凑出 sum/2 的子集。

```python
class Solution:
    def canPartition(self, nums):
        s = sum(nums)
        if s & 1: return False
        target = s // 2
        dp = [False] * (target + 1)
        dp[0] = True
        for x in nums:
            for j in range(target, x - 1, -1):    # 01 背包必须逆序
                dp[j] = dp[j] or dp[j - x]
        return dp[target]
```

### 75. 颜色分类

> [LeetCode 75](https://leetcode.cn/problems/sort-colors/)

#### 🎯 思路（荷兰国旗）

三指针：l 指向 0 区右边界，r 指向 2 区左边界，i 扫描。

```python
class Solution:
    def sortColors(self, nums):
        l, i, r = 0, 0, len(nums) - 1
        while i <= r:
            if nums[i] == 0:
                nums[l], nums[i] = nums[i], nums[l]
                l += 1; i += 1
            elif nums[i] == 2:
                nums[r], nums[i] = nums[i], nums[r]
                r -= 1                              # i 不动！
            else:
                i += 1
```
