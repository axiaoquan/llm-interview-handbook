# 06 · BinaryTree 二叉树

## 🎯 章节核心

### 二叉树的类型
- 满二叉树 / 完全二叉树
- 二叉搜索树（BST） / 平衡二叉搜索树（AVL）

### 三种遍历
| 遍历 | 顺序 | 中节点位置 |
|---|---|---|
| 前序 | **中** 左 右 | 最先 |
| 中序 | 左 **中** 右 | 中间 |
| 后序 | 左 右 **中** | 最后 |

### 递归三要素
1. 确定参数和返回值
2. 确定终止条件
3. 确定单层递归逻辑

### 迭代法
- 前/后序：用栈，**访问和处理同步**
- 中序：需要辅助指针 `cur`，**访问和处理不同步**
- 统一迭代法：用 `None` 标记
- 层序遍历：用队列
