# 01 · Array 数组

## 🎯 章节核心

数组是最基础的数据结构。这一章重点掌握：
- **二分查找** —— 区间定义（左闭右闭 vs 左闭右开）
- **双指针** —— 快慢指针，常用于原地修改
- **滑动窗口** —— 子数组类问题
- **前缀和** —— 区间和 O(1) 查询

## 📌 基础知识

- 数组下标从 0 开始
- 数组内存空间地址连续 → **删除/插入需要移动其他元素，O(n)**
- C++ 中二维数组在地址空间上是连续的
