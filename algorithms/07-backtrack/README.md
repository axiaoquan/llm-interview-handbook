# 07 · Backtrack 回溯

## 🎯 章节核心

**回溯 = 穷举**。本质是把搜索空间画成一棵树，DFS 遍历，过程中尝试剪枝。

适用场景：
- **组合**：N 个数里选 k 个
- **切割**：字符串切分
- **子集**：所有可能的子集
- **排列**：全排列
- **棋盘**：N 皇后 / 数独

### 模板
```cpp
void backtracking(参数) {
    if (终止条件) {
        存放结果;
        return;
    }
    for (选择 : 当前层的选择列表) {
        处理节点;
        backtracking(...);    // 递归
        撤销处理;              // 回溯
    }
}
```
