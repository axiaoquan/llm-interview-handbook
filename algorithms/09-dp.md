## 基础题

### Q01 · 斐波那契数

> [LeetCode 509](https://leetcode.cn/problems/fibonacci-number/) · 难度：⭐

```python
def fib(n):
    if n < 2: return n
    a, b = 0, 1
    for _ in range(n - 1):
        a, b = b, a + b
    return b
```

### Q02 · 爬楼梯

> [LeetCode 70](https://leetcode.cn/problems/climbing-stairs/) · 难度：⭐⭐

每次爬 1 或 2 阶，f(n) = f(n-1) + f(n-2)。

```python
def climbStairs(n):
    if n <= 2: return n
    a, b = 1, 2
    for _ in range(n - 2):
        a, b = b, a + b
    return b
```

### Q04 · 不同路径

> [LeetCode 62](https://leetcode.cn/problems/unique-paths/) · 难度：⭐⭐

`dp[i][j] = dp[i-1][j] + dp[i][j-1]`，第一行第一列全为 1。

```python
def uniquePaths(m, n):
    dp = [[1] * n for _ in range(m)]
    for i in range(1, m):
        for j in range(1, n):
            dp[i][j] = dp[i-1][j] + dp[i][j-1]
    return dp[-1][-1]
```

不同路径 II：有障碍 → 障碍点 dp 设为 0。

### Q05 · 整数拆分

> [LeetCode 343](https://leetcode.cn/problems/integer-break/) · 难度：⭐⭐⭐

**dp[i]**：i 拆分获得的最大乘积。

对 i，定义 j（1 ≤ j < i），分两种情况：
- $j \cdot (i - j)$：拆成两段
- $j \cdot dp[i-j]$：(i-j) 继续拆

$$
dp[i] = \max(j \cdot (i-j),\ j \cdot dp[i-j],\ dp[i])
$$

初始化：`dp[2] = 1`。

```python
def integerBreak(n):
    dp = [0] * (n + 1)
    dp[2] = 1
    for i in range(3, n + 1):
        for j in range(1, i // 2 + 1):
            dp[i] = max(dp[i], j * (i - j), j * dp[i - j])
    return dp[n]
```

## 背包问题

### Q06 · 01 背包（二维 dp）

#### 🎯 题目

n 件物品（每件只能用一次），背包容量 W。第 i 件物品重 weight[i]，价值 value[i]。求装入背包的最大总价值。

#### 📖 五部曲

**1) dp 数组定义**

`dp[i][j]` = 从下标 [0..i] 物品中任取，放进容量 j 的背包，最大价值。

**2) 递推公式**

- **不放物品 i**：`dp[i-1][j]`
- **放物品 i**（前提 j ≥ weight[i]）：`dp[i-1][j - weight[i]] + value[i]`

$$
dp[i][j] = \max(dp[i-1][j],\ dp[i-1][j - w_i] + v_i)
$$

**3) 初始化**

- `dp[i][0] = 0`（容量为 0 装不了）
- `dp[0][j]`：j ≥ weight[0] 时为 value[0]，否则 0

**4) 遍历顺序**

二维 dp 下，**先物品后容量** 或 **先容量后物品** 都行（只依赖左上）。

---

### Q07 · 01 背包（一维滚动数组）

#### 🎯 优化

观察二维 dp，`dp[i][j]` 只依赖**上一行**。把 `dp[i-1]` 拷贝到 `dp[i]` → 一维。

#### 📖 关键：j 必须**逆序**遍历

$$
dp[j] = \max(dp[j],\ dp[j - w_i] + v_i)
$$

**为什么逆序？**

如果**正序**遍历，`dp[j - w_i]` 会被更新成"已经放过物品 i"的值，**导致物品 i 被重复使用**。

逆序 j 则保证 `dp[j - w_i]` 还是上一轮的（未放 i）。

```python
def knapsack01(weights, values, W):
    dp = [0] * (W + 1)
    for i in range(len(weights)):
        for j in range(W, weights[i] - 1, -1):     # 逆序！
            dp[j] = max(dp[j], dp[j - weights[i]] + values[i])
    return dp[W]
```

### Q08 · 完全背包

每件物品**可以无限使用**。

**与 01 背包的区别**：j **正序**遍历（允许重复用同一物品）。

```python
def unbounded_knapsack(weights, values, W):
    dp = [0] * (W + 1)
    for i in range(len(weights)):
        for j in range(weights[i], W + 1):         # 正序！
            dp[j] = max(dp[j], dp[j - weights[i]] + values[i])
    return dp[W]
```

### 📖 求**组合数 vs 排列数**

| 问题 | 遍历顺序 |
|---|---|
| 组合数（选哪些） | **外物品，内容量** |
| 排列数（顺序也算） | **外容量，内物品** |

## 子序列 / 子串

### Q09 · 最长递增子序列

> [LeetCode 300](https://leetcode.cn/problems/longest-increasing-subsequence/) · 难度：⭐⭐⭐

**dp[i]**：以 i 结尾的 LIS 长度。

```python
def lengthOfLIS(nums):
    dp = [1] * len(nums)
    for i in range(len(nums)):
        for j in range(i):
            if nums[j] < nums[i]:
                dp[i] = max(dp[i], dp[j] + 1)
    return max(dp)
```

**O(n²)**。可优化到 O(n log n)：维护一个 tails 数组用二分。

### Q10 · 最长公共子序列

> [LeetCode 1143](https://leetcode.cn/problems/longest-common-subsequence/) · 难度：⭐⭐⭐

**dp[i][j]**：text1 前 i 个字符与 text2 前 j 个字符的 LCS 长度。

$$
dp[i][j] =
\begin{cases}
dp[i-1][j-1] + 1, & \text{if } s_1[i-1] = s_2[j-1] \\
\max(dp[i-1][j], dp[i][j-1]), & \text{otherwise}
\end{cases}
$$

```python
def longestCommonSubsequence(s1, s2):
    m, n = len(s1), len(s2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if s1[i-1] == s2[j-1]:
                dp[i][j] = dp[i-1][j-1] + 1
            else:
                dp[i][j] = max(dp[i-1][j], dp[i][j-1])
    return dp[m][n]
```

### Q11 · 编辑距离

> [LeetCode 72](https://leetcode.cn/problems/edit-distance/) · 难度：⭐⭐⭐⭐

**dp[i][j]**：word1 前 i 字符变到 word2 前 j 字符的最少操作。

$$
dp[i][j] =
\begin{cases}
dp[i-1][j-1], & s_1[i-1] = s_2[j-1] \\
1 + \min(dp[i-1][j],\ dp[i][j-1],\ dp[i-1][j-1]), & \text{else}
\end{cases}
$$

三个 min 分别对应：删除 / 插入 / 替换。

边界：`dp[0][j] = j`，`dp[i][0] = i`。

## 打家劫舍 / 股票

### Q12 · 打家劫舍

> [LeetCode 198](https://leetcode.cn/problems/house-robber/) · 难度：⭐⭐

**dp[i]**：偷到第 i 间能拿到的最大金额。

```python
def rob(nums):
    if not nums: return 0
    if len(nums) == 1: return nums[0]
    dp = [0] * len(nums)
    dp[0], dp[1] = nums[0], max(nums[0], nums[1])
    for i in range(2, len(nums)):
        dp[i] = max(dp[i-1], dp[i-2] + nums[i])
    return dp[-1]
```
